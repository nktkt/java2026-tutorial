# Java Beginner Tutorial

A step-by-step Java tutorial project for learning the fundamentals of Java programming.

## Lessons

| # | Package | Topic |
|---|---------|-------|
| 01 | `lesson01_hello` | Hello World — basic program structure |
| 02 | `lesson02_variables` | Variables & data types (int, double, String, boolean) |
| 03 | `lesson03_operators` | Arithmetic, comparison & logical operators |
| 04 | `lesson04_conditions` | Conditional statements (if / switch / ternary) |
| 05 | `lesson05_loops` | Loops (for, while, do-while, break, continue) |
| 06 | `lesson06_arrays` | Arrays & 2D arrays |
| 07 | `lesson07_methods` | Methods, return values & overloading |
| 08 | `lesson08_classes` | Classes & objects (OOP basics) |
| 09 | `lesson09_inheritance` | Inheritance & polymorphism |
| 10 | `lesson10_interfaces` | Interfaces & abstract classes |

## How to Use

1. Open the project in IntelliJ IDEA.
2. Navigate to `src/` and start from **lesson01**.
3. Read the comments, run each file, and try the practice exercises at the bottom.

## Requirements

- Java 17 or later
- IntelliJ IDEA (recommended)
